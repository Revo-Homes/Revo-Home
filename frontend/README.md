# REVO HOMES

**Find Your Perfect Space** – A modern real estate platform for buying, selling, and renting properties.

## Tech Stack

- React 19 + Vite
- Tailwind CSS 4
- React Router v6

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) (or the port shown).

## Build

```bash
npm run build
npm run preview
```

## Project Structure

```
src/
├── components/       # Reusable UI components
│   ├── Navbar.jsx
│   ├── Footer.jsx
│   ├── SearchBar.jsx
│   ├── PropertyCard.jsx
│   └── FilterSidebar.jsx
├── contexts/         # React contexts (Auth)
├── layouts/          # Page layouts
├── pages/            # Route pages
├── routes/           # App routing
└── index.css         # Global styles + Tailwind
```

## Features

- **Landing Page** – Hero, Featured Properties, Popular Cities, Services, Why Choose, Testimonials
- **Auth** – Login (Email/Phone OTP), OTP Verify, Signup
- **Properties** – Search, Filter, Grid view
- **Property Details** – Full details after login
- **Sell Property** – 5-step listing form
- **User Dashboard** – Profile, Properties, Saved, Enquiries, Visits, Settings
- **Tools** – EMI Calculator, Home Loan Assistance
- **Admin Panel** – Property approval, User management, Analytics
